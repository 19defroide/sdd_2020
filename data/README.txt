OPEN_BIO_2018_7325.csv
----------------------
Source : Améli OpenBio http://open-data-assurance-maladie.ameli.fr/biologie/index.php

Extraction d'une partie des données de remboursement, pour l'acte 7325 « ANTIGENE CA 125 : DOSAGE » (il s'agit d'un dosage de marqueur tumoral).

En-têtes : 
AGE
SEXE
BEN_REG : région bénéficiaire (cf https://www.data.gouv.fr/fr/datasets/nomenclature-code-des-nouvelles-regions-2016/ pour les codes Insee)
DNB : nombre d'actes
REM : montants remboursés
BSE : montants remboursables
